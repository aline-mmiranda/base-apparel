# Base Apparel Coming Soon Page

Projeto desenvolvido como desafio de responsividade e validação de formulário, inspirado no layout do Frontend Mentor.

## Descrição

Esta landing page simula uma tela de "coming soon" para uma loja de moda. O foco do projeto é a responsividade, uso de HTML semântico, CSS moderno e validação de e-mail com feedback visual.

## Funcionalidades

- Layout responsivo para desktop e mobile
- Troca automática da imagem principal conforme o tamanho da tela
- Validação de e-mail com mensagem de erro e feedback visual
- Ícone de erro exibido ao lado do input quando o e-mail é inválido
- Estilização do botão ao ser clicado
- Acessibilidade básica (uso de alt em imagens, foco visual)

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Flexbox, Media Queries, variáveis CSS, unidades rem)
- JavaScript (validação do formulário e manipulação de classes)

## Como usar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Teste a responsividade redimensionando a janela
4. Digite um e-mail inválido para ver o feedback visual
